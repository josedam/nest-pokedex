 # Ejecutar en desarrollo

 1. Descargar la imagen del proyecto
 2. Instalar dependencias
 ```
    yarn install
 ```
 3. Tener el nest CLI instalado
 ```
    npm i -g @nestjs/cli
 ```
 4. Levantar base se datos
 ```
  docker-compose up -d
 ```
